# StockTray v0.2.10

## Changes

- Added volume ratio support from Eastmoney quote field `f10`.
- Added volume ratio as a configurable popup and tray tooltip indicator.
- Fixed tray left-click popup handling on Windows by accepting both mouse down/up tray events with debouncing.
- Hardened popup display by unminimizing and keeping the popup on top before showing it.

## Build

- Windows installer: `StockTray_0.2.10_x64-setup.exe`
