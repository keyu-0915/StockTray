韭菜托盘

一个安静待在 Windows 系统托盘里的 A 股行情与市场风格工具。需要时点一下，快速查看持仓盈亏，以及小登、中登、老登谁更占优。

软件介绍

轻量、常驻、少打扰的股票行情入口

韭菜托盘 StockTray 不试图替代完整证券交易终端。它把常用行情信息收在托盘里，平时不占桌面空间，打开弹窗后用紧凑的 Windows 11 风格界面展示价格、涨跌幅、成交指标、当日盈亏和持仓盈亏。

功能

从持仓速览到市场风格判断

01

托盘常驻

启动后常驻系统托盘，左键打开或关闭行情弹窗，右键进入菜单。

02

行情弹窗

弹窗支持透明背景、圆角、自动消失和鼠标悬停保持，适合快速扫一眼。

03

自选股管理

可配置股票代码、持仓、成本、弹窗显示和托盘提示显示。

04

盈亏速览

托盘状态色会根据总持仓盈亏变化，弹窗中可显示当日和持仓盈亏。

05

指标可选

支持价格、涨跌额、涨跌幅、成交量、成交额、量比、换手率等字段。

06

市场风格

独立评估小登、中登、老登，结合成分收益、宽基指数与数据质量输出倾向。

07

贡献与趋势

查看细分方向、成分股权重与贡献，并用全天时间轴追踪三类风格的盘中变化。

08

本地历史库

按交易日长期归档最终分析和趋势点，在客户端统计占用并按日期管理。

09

多源自动降级

按拖动顺序读取行情，支持东方财富、腾讯和自建富途 OpenD。

小登、中登、老登相互独立，又会竞争有限的市场资金。StockTray 同时观察绝对强弱和相对偏好，能够区分明确占优、共同走强、普遍偏弱与快速轮动。

  • 宽基指数确认与严格质量门槛
  • 细分方向、成分贡献和全天趋势

使用说明

安装后从系统托盘开始使用

  1. 安装并启动 运行安装包，启动后韭菜托盘会出现在 Windows 系统托盘。
  2. 添加自选股 打开设置页，输入股票代码、持仓和成本，保存后开始刷新行情。
  3. 配置显示字段 选择弹窗和托盘提示中要显示的价格、涨跌、成交和盈亏指标。
  4. 快速查看行情 左键托盘图标打开弹窗，快速查看行情和紧凑市场风格卡片。

常见问题

FAQ

韭菜托盘是交易软件吗？

不是。它只用于个人行情查看和持仓盈亏速览，不提供交易能力，也不构成投资建议。

配置文件保存在哪里？

默认保存在 `%APPDATA%\StockTray`。其中包含配置、当日市场快照和长期市场风格历史。

为什么弹窗里没有股票？

请先在设置页添加自选股，并确认对应股票勾选了“弹窗”显示。

可以商用或二次分发吗？

可以，但项目采用 GPL-3.0-or-later。分发修改版或衍生版本时，需要遵守 GPL 的源码开放和再分发条款。

隐私说明

本地配置，不做账号体系

韭菜托盘不提供注册登录功能。自选股、持仓、成本和显示偏好保存在本机配置文件中。应用会为了获取行情访问公开行情接口，不会把你的本地配置上传到本项目服务器。

请注意，行情接口提供方可能会看到网络请求所需的基础信息，例如 IP 地址和请求时间。
