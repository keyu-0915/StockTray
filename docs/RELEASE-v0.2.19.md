# StockTray v0.2.19

This release turns market style into a first-class StockTray workflow.

It introduces scientifically gated young, middle, and old style analysis with exact samples, float-cap weighting, broad-index confirmation, selectable stock and subsector attribution, a full-session intraday trend, and a compact popup summary.

Market-style results now form a local long-term history library instead of being deleted after one trading day. The client reports stored days, trend points, and disk usage, and supports date-level cleanup.

The release also adds ordered quote-source fallback, a native Futu OpenD adapter with Linux Docker deployment files, and a broad UI/UX refinement across borderless windows, responsive layouts, themes, transitions, popup sizing, auto-hide behavior, and support QR presentation.
