# StockTray v0.2.13

## Changes

- Added free watchlist ordering with per-row move up and move down controls.
- Added one-click sorting by holdings and quote change percent.
- Sorting buttons toggle between descending and ascending order.
- The saved watchlist order is reused by the popup and tray tooltip data.

## Build

- Windows installer: `StockTray_0.2.13_x64-setup.exe`
